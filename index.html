环境要求

macOS

  • macOS 10.15 或更高版本
  • Node.js 16+ 或 npm
  • Xcode Command Line Tools
  • Homebrew (推荐)

Windows

  • Windows 10/11
  • Node.js 16+ 或 npm
  • Windows Terminal (推荐)
  • PowerShell 5.1+

Linux

  • Ubuntu 18.04+ / CentOS 7+
  • Node.js 16+ 或包管理器
  • curl 或 wget
  • 基础构建工具

安装步骤

使用 NPM 安装 (推荐)

Terminal 
# 全局安装
npm install -g @anthropic/claude-code

# 验证安装
claude-code --version

二进制文件安装

macOS

Terminal 
# 使用 Homebrew
brew install anthropic/tap/claude-code

# 或手动下载
curl -L https://github.com/anthropics/claude-code/releases/latest/download/claude-code-macos -o claude-code
chmod +x claude-code
sudo mv claude-code /usr/local/bin/

Windows

PowerShell 
# 使用 PowerShell
Invoke-WebRequest -Uri "https://github.com/anthropics/claude-code/releases/latest/download/claude-code-windows.exe" -OutFile "claude-code.exe"

# 添加到 PATH 或移动到系统目录
Move-Item claude-code.exe $env:USERPROFILE\AppData\Local\Microsoft\WindowsApps\
Windows 特别说明：首次运行可能需要允许 Windows Defender 执行。

Linux

Terminal 
# 下载并安装
curl -L https://github.com/anthropics/claude-code/releases/latest/download/claude-code-linux -o claude-code
chmod +x claude-code
sudo mv claude-code /usr/local/bin/

# 或使用包管理器 (Ubuntu/Debian)
wget -qO- https://github.com/anthropics/claude-code/releases/latest/download/claude-code_amd64.deb
sudo dpkg -i claude-code_amd64.deb

从源码编译

Terminal 
# 克隆仓库
git clone https://github.com/anthropics/claude-code.git
cd claude-code

# 安装依赖
npm install

# 构建
npm run build

# 链接到全局
npm link

配置设置

1

获取 API 密钥

首先需要在 Anthropic 注册并获取 API 密钥：

注册获取 API 密钥
2

配置环境变量

配置文件 
# 临时设置 (当前会话)
export ANTHROPIC_API_KEY="your-api-key-here"

# 永久设置 (添加到 ~/.bashrc 或 ~/.zshrc)
echo 'export ANTHROPIC_API_KEY="your-api-key-here"' >> ~/.bashrc
source ~/.bashrc

Windows 设置

# PowerShell
$env:ANTHROPIC_API_KEY="your-api-key-here"

# 永久设置
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "your-api-key-here", "User")
3

初始化配置

Terminal 
# 初始化配置
claude-code init

# 验证配置
claude-code --help

# 测试连接
claude-code "Hello, Claude!"

常见问题与故障排除

安装后找不到 claude-code 命令

解决方案：

  • 检查 PATH 环境变量是否包含安装目录
  • 重新启动终端或重新加载配置文件
  • 确认安装过程没有错误
# 检查 PATH
echo $PATH

# macOS/Linux 重新加载配置
source ~/.bashrc  # 或 ~/.zshrc
API 密钥配置错误

检查步骤：

  • 确认 API 密钥格式正确
  • 检查环境变量是否正确设置
  • 验证网络连接是否正常
# 检查环境变量
echo $ANTHROPIC_API_KEY

# 测试连接
claude-code "test connection"
Windows 权限问题

Windows 特有解决方案：

  • 以管理员身份运行 PowerShell
  • 检查 Windows Defender 设置
  • 确认执行策略允许运行脚本
# 检查执行策略
Get-ExecutionPolicy

# 设置执行策略 (如需要)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
